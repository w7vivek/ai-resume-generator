# AI Resume Generator

This is a full-stack resume generator project where users can create a resume summary using AI and download their resume for free.

I built this project to understand how frontend and backend work together in a real application.

---

## Features

- Generates a resume summary using AI
- Simple and clean user interface
- Resume can be downloaded for free
- Frontend and backend are properly connected

---

## Tech Used

### Frontend
- React
- JavaScript
- HTML
- CSS

### Backend
- Node.js
- Express.js

### Hosting
- Backend is hosted on Render

---

## How the Project Works

1. User fills in their details on the website  
2. Data is sent from React frontend to the backend API  
3. Backend processes the data and generates an AI summary  
4. The generated resume is sent back to the frontend  
5. User can download the resume
